# Hospital Readmission Risk Analysis

## Overview
This is a project that analyzes hospital data to identify factors associated with 30-day readmission risks.

## Objective
- ID key drivers of reamission
- Engineer meaningful clinical features
- Develop a simple risk scoring approach

## Tools Used
- python (pandas, matplotlib)
- Jupyter notebook

## Key Findings
- Longer hospital admissions lead to an increased readmission risk
- High medication burden correlates with higher risk
- Composite risk scoring shows strong stratification ability

## Project Structure
- notebooks/ -> analysis notebook
- data/ -> dataset
